# Swarm Codex

This folder records the working specification for a local, inspectable swarm system supervised by Codex.

The short version:

```text
Outer Codex conversation = supervisor
Local worker instances = bounded agents
LM Studio = local model backend
Swarm dashboard = inspectable nested workspace
```
